# Jagruthi Pulumati - DVA Portfolio

Static HTML/CSS portfolio for DVA projects:

- **Impact of COVID-19 on Maternal & Newborn Health**
- **Bank Marketing Campaign Analysis**

## Files

- `index.html` - portfolio landing page with project card and dashboard showcase
- `project.html` - detailed maternal and newborn health project explanation
- `bank-marketing.html` - detailed bank marketing project explanation
- `styles.css` - all styling
- `assets/dashboard.png` - project dashboard screenshot
- `assets/bank-*.jpeg` - Tableau dashboard screenshots for the bank marketing project

## Deployment

This site has no build step. Deploy the full folder as a static site.

Good options:

- GitHub Pages
- Netlify
- Vercel static project

The homepage is `index.html`.
